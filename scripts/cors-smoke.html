vucible CORS Smoke Test

Tests whether OpenAI and Gemini image generation APIs accept browser-origin calls. Open DevTools Network tab before running. Results determine architecture (§9.6 / §14.T).

API Keys

OpenAI test generates one tiny image (~$0.02 at 256x256). Gemini list-models is free. Gemini generateContent with image output may cost depending on your tier.

Note: OpenAI returns CORS headers on success but NOT on 401 errors (Cloudflare edge strips them). If your key is wrong, you'll see a generic "Failed to fetch" instead of a clean error — that's expected, not a CORS block.

OpenAI

Test 1: POST /v1/images/generations (minimal body)
Test 2: GET /v1/models (list models — free, auth check)

Gemini

Test 3: GET /v1/models (list models — free)
Test 4: POST generateContent — text-only (image generation)
